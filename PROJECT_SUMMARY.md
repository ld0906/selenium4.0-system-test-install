# 大牛测试系统 - Python版项目总结

## 🎯 项目概述

本项目是对原Java SpringBoot若依(RuoYi)管理系统的Python Flask重构版本，保持了相同的功能和UI设计，但使用了更轻量级的技术栈，便于快速部署和测试。

## 📊 技术对比

| 组件 | 原系统(Java) | 新系统(Python) |
|------|-------------|---------------|
| 后端框架 | Spring Boot | Flask 3.0 |
| 数据库 | MySQL | SQLite3 |
| ORM | MyBatis | SQLAlchemy |
| 模板引擎 | Thymeleaf | Jinja2 |
| 权限管理 | Apache Shiro | Flask-Login + 自定义装饰器 |
| 前端 | Bootstrap + jQuery | Bootstrap + jQuery |
| 构建工具 | Maven | pip |
| 启动方式 | java -jar | python run.py |

## ⚠️ 注意事项

1. **SQLite限制**: 不适合高并发场景，大型项目建议迁移到MySQL/PostgreSQL
2. **静态资源**: 需要配置CDN或从原系统复制静态文件
3. **生产环境**: 需要配置Gunicorn+Nginx，不要直接使用Flask开发服务器
4. **安全配置**: 修改SECRET_KEY和默认密码
